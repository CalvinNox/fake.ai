# Calvin 鉴定所 H5 小游戏

一个纯前端 H5 互动小游戏，可直接部署到 GitHub Pages。

## 文件说明

- `index.html`：页面入口
- `styles.css`：界面样式和动画
- `script.js`：交互逻辑

## 本地预览

直接用浏览器打开 `index.html` 即可预览。

## GitHub Pages 部署

1. 新建 GitHub 仓库。
2. 上传本目录中的 `index.html`、`styles.css`、`script.js` 和 `README.md`。
3. 进入仓库 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`，Branch 选择 `main` 和 `/root`。
5. 保存后等待 GitHub Pages 生成访问链接。
